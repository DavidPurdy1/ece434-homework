# Etch-e-Sketch

## Overview

Move around the screen and draw a picture! Erase the etch-a-sketch by pressing 'c' and start over.

---

Example command to run

```bash
./etch-e-sketch.py 8 8
``` 
Example Output:

```
Welcome to the etch-a-sketch simulator!
Commands:
  u - move up
  d - move down
  l - move left
  r - move right
  c - clear the screen
  q - quit
```
